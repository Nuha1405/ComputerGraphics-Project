# ComputerGraphics-Project
The proposed system will be a 2D graphical simulation that includes:

•	City background with sky, sun, moon, and clouds
•	Roads with zebra crossing and footpaths
•	Buildings, trees, lamps, and benches
•	Moving vehicles such as cars and buses
•	Pedestrian movement across the road
•	Traffic signals for road management
•	Basic user interaction using keyboard input
